Nâng cấp từ ptit-landing-page.

Các tính năng mới:
- Form đã hoạt động.
- Có đăng nhập để vào trang quản lý.
- Trang quản lý có thống kê, xem, thêm, sửa, xóa sinh viên.

Các công nghệ sử dụng: HTML, CSS, PHP, Bootstrap, JS, AJAX.
