# Student Management System (Web QLSV)

This project is a lightweight web-based application designed to manage student records efficiently. It provides an administrative dashboard for viewing, adding, editing, and deleting student information using PHP and MySQL.

## Project Overview

The **Student Management System** is built to handle academic data with a focus on simplicity. It features a secure login system for administrators and an AJAX-driven interface for managing student data without reloading the page.

### Key Features

* **Authentication:** Secure Admin login system.
* **Dashboard:** Centralized view of all student records.
* **CRUD Operations:**
    * **Create:** Add new students with details (ID, Name, DOB, Email, Phone, Major, Class, CPA).
    * **Read:** View student lists with pagination or filtering (implied by structure).
    * **Update:** Edit existing student information via modal forms.
    * **Delete:** Remove student records from the database.
* **Asynchronous Processing:** Uses AJAX for smoother form submissions and data handling.

## Technology Stack

* **Backend:** PHP (Native)
* **Database:** MySQL / MariaDB
* **Frontend:** HTML5, CSS3, JavaScript (AJAX)
* **Server Environment:** XAMPP / WAMP / LAMP Stack

## Prerequisites

Before running the project, ensure you have the following installed:

* A local web server (e.g., [XAMPP](https://www.apachefriends.org/index.html) or WAMP).
* PHP 8.0 or higher.
* MySQL Database server.
